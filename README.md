# TypeScript Task Manager

A small task manager built with TypeScript and Tailwind CSS.

## Features
- Add tasks
- Mark tasks as done
- Delete tasks

## Tech Stack
- TypeScript
- HTML
- Tailwind CSS

## How to Run
1. Install dependencies:
   ```bash
   npm install
   ```
2. Build the project
   ```bash
    npx tsc
   ```
3. Open index.html in browser
